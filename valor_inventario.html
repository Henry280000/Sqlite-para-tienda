Consultas GROUP BY y HAVING

Esta página muestra el valor total del inventario agrupado por categoría, utilizando las cláusulas GROUP BY y HAVING.

SQL usado: SELECT c.nombre as categoria, SUM(p.cantidad * p.precio) as valor_total FROM productos p JOIN categorias c ON p.categoria_id = c.id GROUP BY p.categoria_id HAVING valor_total > 0 ORDER BY valor_total DESC
